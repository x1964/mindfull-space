# 🧠 MindfulSpace Pro

تطبيق ويب تفاعلي حديث يهدف إلى تحسين الإنتاجية وتنظيم الأفكار من خلال لوحة عمل لا نهائية (Infinite Canvas) تجمع بين البساطة والقوة.

---

## 🚀 الفكرة الأساسية

**MindfulSpace Pro** هو مساحة رقمية مرنة تتيح لك:

- كتابة ملاحظات سريعة (Sticky Notes)
- إنشاء قوائم مهام (Todo Lists)
- استخدام مؤقت بومودورو للتركيز
- تشغيل ساعة إيقاف لتتبع الوقت

كل ذلك داخل **لوحة لا نهائية قابلة للحركة والتكبير**.

---

## 🎨 التصميم (UI/UX)

- 🌞 وضع فاتح مريح للعين (Light Mode)
- 🎨 خلفية متدرجة (أبيض → رمادي فاتح)
- 🧊 بطاقات Glassmorphism (شفافة + Blur)
- ✨ أنيميشن ناعمة وسلسة
- 🔤 خط عربي احترافي (Tajawal)
- ↔️ دعم كامل للغة العربية (RTL)

---

## ⚙️ المميزات

### 🧭 لوحة لا نهائية (Infinite Canvas)
- تحريك (Pan) باستخدام الماوس
- تكبير وتصغير (Zoom) بعجلة الفأرة
- حفظ تلقائي لموضع الكاميرا

---

### 🧩 إدارة العناصر
- Drag & Drop للبطاقات
- حرية كاملة في ترتيب العناصر
- تكبير العنصر النشط (Focus Effect)
- حذف مع رسالة تأكيد

---

### 📝 الملاحظات (Notes)
- Editable Text Area
- حفظ تلقائي أثناء الكتابة
- Placeholder عند الفراغ

---

### ✅ قائمة المهام (Todo List)
- إضافة مهام عبر Popup
- تحديد المهام المنجزة ✔️
- حذف مهام فردية
- رسالة عند عدم وجود مهام

---

### 🍅 مؤقت بومودورو (Pomodoro Timer)
- مدة 25 دقيقة للتركيز
- تشغيل / إيقاف / إعادة تعيين
- تنبيه عند الانتهاء
- تحديث لحظي للوقت

---

### ⏱️ ساعة الإيقاف (Stopwatch)
- عداد تصاعدي (ساعات / دقائق / ثواني)
- تحكم كامل في التشغيل

---

## 💾 التخزين والبيانات

- 💡 حفظ تلقائي كل 2 ثانية باستخدام `localStorage`
- 🔄 استرجاع البيانات بعد إعادة تحميل الصفحة
- 📤 تصدير البيانات كملف JSON
- 🗑️ مسح كامل مع تأكيد

---

## 🎮 واجهة المستخدم

### 🔝 الشريط العلوي (Navbar)
- Reset View → إعادة ضبط العرض
- Export → تصدير البيانات
- Clear All → مسح الكل

### ➕ القائمة العائمة (FAB Menu)
- ملاحظة جديدة
- قائمة مهام
- بومودورو
- ساعة إيقاف
